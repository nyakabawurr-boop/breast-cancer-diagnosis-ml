# 5-Minute Video Script

## 1. Introduction
Hi, I’m Raymond Royal Nyakabawu. In this project, I used the Breast Cancer Wisconsin dataset to predict whether tumors are benign or malignant using machine learning.

## 2. Business Problem
The goal was not just to classify tumors, but to think carefully about the risk of missing malignant cases. In healthcare, false negatives can be especially costly, so this project focused on that business and clinical risk.

## 3. Data Understanding and EDA
I began by exploring the dataset, checking the target distribution, reviewing correlations between features, and comparing key variables such as perimeter, radius, and area across benign and malignant diagnoses.

## 4. Modeling Approach
I built and compared two machine learning models: a Decision Tree and a Random Forest. I used feature scaling, a train-test split, confusion matrices, ROC curves, Precision-Recall curves, and feature importance analysis to evaluate performance.

## 5. Results
The Random Forest model outperformed the Decision Tree and delivered a stronger baseline for this classification task. It made fewer classification errors and showed stronger overall performance based on the evaluation metrics.

## 6. Recommendation
My recommendation is to use Random Forest as the stronger baseline model for this project. However, this should be treated as a decision-support tool and not as a replacement for clinical diagnosis.

## 7. Closing
This project helped me strengthen my Python, machine learning, data visualization, and business communication skills. Thank you for watching.
