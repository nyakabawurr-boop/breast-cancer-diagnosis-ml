# Project Summary

## Objective
To predict whether a breast tumor is benign or malignant using machine learning and compare Decision Tree and Random Forest models.

## Business Importance
In healthcare, missing a malignant tumor is more costly than incorrectly flagging a benign one. This project focuses on model performance with that risk in mind.

## Methods Used
- Data cleaning
- Exploratory data analysis
- Feature scaling
- Decision Tree
- Random Forest
- Confusion matrix
- ROC curve
- Precision-Recall curve
- Feature importance

## Key Result
The Random Forest model performed better than the Decision Tree and provided a stronger baseline for identifying malignant cases.

## Recommendation
Use Random Forest as the stronger baseline model, while treating the model as a decision-support tool rather than a replacement for clinical diagnosis.
